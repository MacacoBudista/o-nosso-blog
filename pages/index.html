Um Macaco atrás da Tela

O objectivo deste blog é apresentar algumas soluções das aplicações mais usadas, tais como a configuração e personalização das mesmas, usando o terminal no Linux.

  • Editor de Texto:
  • Vim

  • Gestor de ficheiros:
  • Vifm

  • Gestor de sessões virtual de terminais:
  • Tmux

  • Cliente para gerir o progresso do código:
  • Git

  • Cliente de Email:
  • Mutt

  • Gestor do protocolo de email IMAP:
  • Isync

  • Gestor do protocolo de email SMTP:
  • Msmtp

  • Cliente de FTP:
  • Lftp

  • Cliente Web:
  • W3m

  • Configuração do audio:
  • Alsa

  • Gestor de ligações wifi:
  • WPA
